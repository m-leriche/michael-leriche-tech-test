# Michael le Riche – Venn Tech Test

A tech test project scaffolded with Vite, React, TailwindCSS, and TypeScript.

## 🛠️ Tech Stack

- **Framework:** React (via Vite)
- **Styling:** TailwindCSS
- **Language:** TypeScript
- **Testing:** Vitest + Testing Library

## 🚀 Getting Started

### Install dependencies

```bash
npm install
```

### Run Local Environment 

```bash
npm run dev
```

### Run Tests 

```bash
npm run test
```
